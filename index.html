Vidu ツール取扱説明書

🎬 1. Start Frame & End Frame（Image to Video）

概要: 1枚または2枚の画像を基に、AIがその間の動きを補完して動画を生成します。

  • Start Frame: 動画の開始時の画像。
  • End Frame: 動画の終了時の画像。

活用例:

  • キャラクターが歩き出すシーンから目的地に到着するシーンまでの移動を表現。
  • 風景の時間経過（朝から夕方）を表現。

ポイント:

  • 「Add End Frame」機能で、2枚の画像間の動きを補完。
  • 「Swap image order」で順序を逆にすることも可能。

参考情報:

  • Viduの【Image to Video】実践ガイド
  • Viduの使い方（高画質化）

🎥 2. Reference to Video

概要: 複数の画像やプロンプトを参照し、より詳細な動きやシーンを指定して動画を生成します。

  • キャラクターと背景を別々に指定。
  • キャラクターの一貫性ある動作。
  • 特定の部位（顔など）だけを参照。

参考情報:

  • Reference to Video活用法
  • Viduの使い方（高画質化）

🎞️ 3. First-to-Last Frame（Vidu Q1）

概要: 複数シーンの開始・終了フレームをつなげて連続動画を生成。

活用例: 猫が草むら → 海辺 → 夕焼け、などのストーリー作成。

参考情報:

  • Vidu Q1入門

✏️ プロンプトの工夫

  • 動作や環境を具体的に描写（例: 雪の積もった冬の森）
  • 視点やカメラワークの明示
  • 英語プロンプトを活用すると精度が向上する傾向

参考: Text to Video活用のコツ

🖼️ 画像選定のポイント

  • キャラクターと背景を別々に設定
  • 画像の一部のみを参照指定可能

参考: Reference活用法
